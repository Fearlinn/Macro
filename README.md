## ✨ Özellikler

- **⚡ Çift Kanallı Makro:** Sol ve Sağ tık için birbirinden bağımsız iş parçacıkları (threading).
- **🎯 Akıllı Odaklama:** Sadece oyun penceresi (CraftRise) açıkken çalışır, masaüstünde hata yapmaz.
- **🎨 Modern UI/UX:** Consolas yazı tipi ve neon renk paletiyle tasarlanmış kullanıcı dostu arayüz.
- **🔄 Otomatik Güncelleme:** GitHub üzerinden sürüm kontrolü yapar ve kendini otomatik olarak günceller.
- **🛠️ Esnek CPS:** 1'den 20'ye kadar anlık ayarlanabilir tıklama hızı.

---

## ⌨️ Kontrol Kısayolları

| Tuş | Fonksiyon |
| :--- | :--- |
| **`X`** | Makroyu Duraklat / Devam Ettir |
| **`F1`** | Uygulamadan Güvenli Çıkış |
| **`Sol/Sağ Tık`** | Basılı tutulduğunda aktifleşir |
